Section 3.2Calculus of Vector-Valued Functions

To study the calculus of vector-valued functions, we follow a similar path to the one we took in studying real-valued functions. First, we define the derivative, then we examine applications of the derivative, then we move on to defining integrals. However, we will find some interesting new ideas along the way as a result of the vector nature of these functions and the properties of space curves.

Derivatives of Vector-Valued Functions

Now that we have seen what a vector-valued function is and how to take its limit, the next step is to learn how to differentiate a vector-valued function. The definition of the derivative of a vector-valued function is nearly identical to the definition of a real-valued function of one variable. However, because the range of a vector-valued function consists of vectors, the same is true for the range of the derivative of a vector-valued function.

Definition

The derivative of a vector-valued function \(\text{r}(t)\) is

\[{r}^{\prime }(t)=\underset{\Delta t\to 0}{\text{lim}}\frac{\text{r}(t+\Delta t)-\text{r}(t)}{\Delta t},\]

provided the limit exists. If \({r}^{\prime }(t)\) exists, then r is differentiable at t. If \({r}^{\prime }(t)\) exists for all t in an open interval \((a,b),\) then r is differentiable over the interval \((a,b).\) For the function to be differentiable over the closed interval \([a,b],\) the following two limits must exist as well:

\[{r}^{\prime }(a)=\underset{\Delta t\to {0}^{+}}{\text{lim}}\frac{\text{r}(a+\Delta t)-\text{r}(a)}{\Delta t}\,\text{and}\,{r}^{\prime }(b)=\underset{\Delta t\to {0}^{-}}{\text{lim}}\frac{\text{r}(b+\Delta t)-\text{r}(b)}{\Delta t}.\]

Many of the rules for calculating derivatives of real-valued functions can be applied to calculating the derivatives of vector-valued functions as well. Recall that the derivative of a real-valued function can be interpreted as the slope of a tangent line or the instantaneous rate of change of the function. The derivative of a vector-valued function can be understood to be an instantaneous rate of change as well; for example, when the function represents the position of an object at a given point in time, the derivative represents its velocity at that same point in time.

We now demonstrate taking the derivative of a vector-valued function.

Example 1

Use the definition to calculate the derivative of the function

\[\text{r}(t)=(3t+4)\,\text{i}+({t}^{2}-4t+3)\,\text{j}.\]

Substitute \(\text{r}(t+\Delta t)\) and \(\text{r}(t)\) into the difference quotient, expand each component, then take the limit as \(\Delta t\to 0.\)

Let’s use (see original):

\[\begin{array}{ll}{r}^{\prime }(t) & =\underset{\Delta t\to 0}{\text{lim}}\frac{\text{r}(t+\Delta t)-\text{r}(t)}{\Delta t} \\ & =\underset{\Delta t\to 0}{\text{lim}}\frac{[(3(t+\Delta t)+4)\,\text{i}+({(t+\Delta t)}^{2}-4(t+\Delta t)+3)\,\text{j}]-[(3t+4)\,\text{i}+({t}^{2}-4t+3)\,\text{j}]}{\Delta t} \\ & =\underset{\Delta t\to 0}{\text{lim}}\frac{(3t+3\Delta t+4)\,\text{i}-(3t+4)\,\text{i}+({t}^{2}+2t\Delta t+{(\Delta t)}^{2}-4t-4\Delta t+3)\,\text{j}-({t}^{2}-4t+3)\,\text{j}}{\Delta t} \\ & =\underset{\Delta t\to 0}{\text{lim}}\frac{(3\Delta t)\,\text{i}+(2t\Delta t+{(\Delta t)}^{2}-4\Delta t)\,\text{j}}{\Delta t} \\ & =\underset{\Delta t\to 0}{\text{lim}}(3\,\text{i}+(2t+\Delta t-4)\,\text{j}) \\ & =3\,\text{i}+(2t-4)\,\text{j}.\end{array}\]

Notice that in the calculations in Example 1, we could also obtain the answer by first calculating the derivative of each component function, then putting these derivatives back into the vector-valued function. This is always true for calculating the derivative of a vector-valued function, whether it is in two or three dimensions. We state this in the following theorem. The proof of this theorem follows directly from the definitions of the limit of a vector-valued function and the derivative of a vector-valued function.

Differentiation of Vector-Valued Functions

Let f, g, and h be differentiable functions of t.

  • If \(\text{r}(t)=f(t)\,\text{i}+g(t)\,\text{j},\) then \({r}^{\prime }(t)={f}^{\prime }(t)\,\text{i}+{g}^{\prime }(t)\,\text{j}.\)
  • If \(\text{r}(t)=f(t)\,\text{i}+g(t)\,\text{j}+h(t)\,\text{k},\) then \({r}^{\prime }(t)={f}^{\prime }(t)\,\text{i}+{g}^{\prime }(t)\,\text{j}+{h}^{\prime }(t)\,\text{k}.\)
Example 2

Use (see original) to calculate the derivative of each of the following functions.

  • \(\text{r}(t)=(6t+8)\,\text{i}+(4{t}^{2}+2t-3)\,\text{j}\)
  • \(\text{r}(t)=3\,\text{cos}\,t\,\text{i}+4\,\text{sin}\,t\,\text{j}\)
  • \(\text{r}(t)={e}^{t}\text{sin}\,t\,\text{i}+{e}^{t}\text{cos}\,t\,\text{j}-{e}^{2t}\,\text{k}\)

Differentiate each component function separately, then recombine the results into the derivative vector.

We use (see original) and what we know about differentiating functions of one variable.

  • The first component of \(\text{r}(t)=(6t+8)\,\text{i}+(4{t}^{2}+2t-3)\,\text{j}\) is \(f(t)=6t+8.\) The second component is \(g(t)=4{t}^{2}+2t-3.\) We have \({f}^{\prime }(t)=6\) and \({g}^{\prime }(t)=8t+2,\) so the theorem gives \({r}^{\prime }(t)=6\,\text{i}+(8t+2)\,\text{j}.\)
  • The first component is \(f(t)=3\,\text{cos}\,t\) and the second component is \(g(t)=4\,\text{sin}\,t.\) We have \({f}^{\prime }(t)=-3\,\text{sin}\,t\) and \({g}^{\prime }(t)=4\,\text{cos}\,t,\) so we obtain \({r}^{\prime }(t)=-3\,\text{sin}\,t\,\text{i}+4\,\text{cos}\,t\,\text{j}.\)
  • The first component of \(\text{r}(t)={e}^{t}\text{sin}\,t\,\text{i}+{e}^{t}\text{cos}\,t\,\text{j}-{e}^{2t}\,\text{k}\) is \(f(t)={e}^{t}\text{sin}\,t,\) the second component is \(g(t)={e}^{t}\text{cos}\,t,\) and the third component is \(h(t)=-{e}^{2t}.\) We have \({f}^{\prime }(t)={e}^{t}(\text{sin}\,t+\text{cos}\,t),\) \({g}^{\prime }(t)={e}^{t}(\text{cos}\,t-\text{sin}\,t),\) and \({h}^{\prime }(t)=-2{e}^{2t},\) so the theorem gives \({r}^{\prime }(t)={e}^{t}(\text{sin}\,t+\text{cos}\,t)\,\text{i}+{e}^{t}(\text{cos}\,t-\text{sin}\,t)\,\text{j}-2{e}^{2t}\,\text{k}.\)

We can extend to vector-valued functions the properties of the derivative that we presented in the Introduction to Derivatives. In particular, the constant multiple rule, the sum and difference rules, the product rule, and the chain rule all extend to vector-valued functions. However, in the case of the product rule, there are actually three extensions: (1) for a real-valued function multiplied by a vector-valued function, (2) for the dot product of two vector-valued functions, and (3) for the cross product of two vector-valued functions.

Properties of the Derivative of Vector-Valued Functions

Let r and u be differentiable vector-valued functions of t, let f be a differentiable real-valued function of t, and let c be a scalar.

\[\begin{array}{lllllll}\text{i.} & & \frac{d}{dt}[c\,\text{r}(t)] & = & c\,{r}^{\prime }(t) & & \text{Scalar multiple} \\ \text{ii.} & & \frac{d}{dt}[\,\text{r}(t)\pm \,\text{u}(t)] & = & {r}^{\prime }(t)\pm \,{u}^{\prime }(t) & & \text{Sum and difference} \\ \text{iii.} & & \frac{d}{dt}[f(t)\,\text{u}(t)] & = & {f}^{\prime }(t)\,\text{u}(t)+f(t)\,{u}^{\prime }(t) & & \text{Scalar product} \\ \text{iv.} & & \frac{d}{dt}[\,\text{r}(t)·\,\text{u}(t)] & = & {r}^{\prime }(t)·\,\text{u}(t)+\,\text{r}(t)·\,{u}^{\prime }(t) & & \text{Dot product} \\ \text{v.} & & \frac{d}{dt}[\,\text{r}(t)\,\times \,\text{u}(t)] & = & {r}^{\prime }(t)\,\times \,\text{u}(t)+\,\text{r}(t)\,\times \,{u}^{\prime }(t) & & \text{Cross product} \\ \text{vi.} & & \frac{d}{dt}[\,\text{r}(f(t))] & = & {r}^{\prime }(f(t))·{f}^{\prime }(t) & & \text{Chain rule} \\ \text{vii.} & & \text{If}\,\text{r}(t)·\,\text{r}(t) & = & c,\,\text{then}\,\text{r}(t)·\,{r}^{\prime }(t)=0. & & \end{array}\]

Proof

The proofs of the first two properties follow directly from the definition of the derivative of a vector-valued function. The third property can be derived from the first two properties, along with the product rule from the Introduction to Derivatives. Let \(\text{u}(t)=g(t)\,\text{i}+h(t)\,\text{j}.\) Then

\[\begin{array}{ll}\frac{d}{dt}[f(t)\,\text{u}(t)] & =\frac{d}{dt}[f(t)(g(t)\,\text{i}+h(t)\,\text{j})] \\ & =\frac{d}{dt}[f(t)g(t)\,\text{i}+f(t)h(t)\,\text{j}] \\ & =\frac{d}{dt}[f(t)g(t)]\,\text{i}+\frac{d}{dt}[f(t)h(t)]\,\text{j} \\ & =({f}^{\prime }(t)g(t)+f(t){g}^{\prime }(t))\,\text{i}+({f}^{\prime }(t)h(t)+f(t){h}^{\prime }(t))\,\text{j} \\ & ={f}^{\prime }(t)\,\text{u}(t)+f(t)\,{u}^{\prime }(t).\end{array}\]

To prove property iv. let \(\text{r}(t)={f}_{1}(t)\,\text{i}+{g}_{1}(t)\,\text{j}\) and \(\text{u}(t)={f}_{2}(t)\,\text{i}+{g}_{2}(t)\,\text{j}.\) Then

\[\begin{array}{ll}\frac{d}{dt}[\,\text{r}(t)·\,\text{u}(t)] & =\frac{d}{dt}[{f}_{1}(t){f}_{2}(t)+{g}_{1}(t){g}_{2}(t)] \\ & ={f}_{1}{}^{\prime }(t){f}_{2}(t)+{f}_{1}(t){f}_{2}{}^{\prime }(t)+{g}_{1}{}^{\prime }(t){g}_{2}(t)+{g}_{1}(t){g}_{2}{}^{\prime }(t) \\ & ={f}_{1}{}^{\prime }(t){f}_{2}(t)+{g}_{1}{}^{\prime }(t){g}_{2}(t)+{f}_{1}(t){f}_{2}{}^{\prime }(t)+{g}_{1}(t){g}_{2}{}^{\prime }(t) \\ & =({f}_{1}{}^{\prime }\,\text{i}+{g}_{1}{}^{\prime }\,\text{j})·({f}_{2}\,\text{i}+{g}_{2}\,\text{j})+({f}_{1}\,\text{i}+{g}_{1}\,\text{j})·({f}_{2}{}^{\prime }\,\text{i}+{g}_{2}{}^{\prime }\,\text{j}) \\ & ={r}^{\prime }(t)·\,\text{u}(t)+\,\text{r}(t)·\,{u}^{\prime }(t).\end{array}\]

The proof of property v. is similar to that of property iv. Property vi. can be proved using the chain rule. Last, property vii. follows from property iv:

\[\begin{array}{lll}\frac{d}{dt}[\,\text{r}(t)·\,\text{r}(t)] & = & \frac{d}{dt}[c] \\ {r}^{\prime }(t)·\,\text{r}(t)+\,\text{r}(t)·\,{r}^{\prime }(t) & = & 0 \\ 2\,\text{r}(t)·\,{r}^{\prime }(t) & = & 0 \\ \text{r}(t)·\,{r}^{\prime }(t) & = & 0.\end{array}\]

  • \(\frac{d}{dt}[\,\text{r}(t)·\,\text{u}(t)]\)
  • \(\frac{d}{dt}[\,\text{u}(t)\,\times \,{u}^{\prime }(t)]\)

    Recall that the cross product of any vector with itself is zero. Furthermore, \(\text{u}\prime\prime (t)\) represents the second derivative of \(\text{u}(t)\text{:}\)

    Therefore,

    \[\frac{d}{dt}[\,\text{u}(t)\,\times \,{u}^{\prime }(t)]={u}^{\prime }(t)\,\times \,{u}^{\prime }(t)+\,\text{u}(t)\,\times \,\text{u}\prime\prime (t).\]

    Recall that the cross product of any vector with itself is zero. Furthermore, \(\text{u}\prime\prime (t)\) represents the second derivative of \(\text{u}(t)\text{:}\)

Tangent Vectors and Unit Tangent Vectors

Recall from the Introduction to Derivatives that the derivative at a point can be interpreted as the slope of the tangent line to the graph at that point. In the case of a vector-valued function, the derivative provides a tangent vector to the curve represented by the function. Consider the vector-valued function \(\text{r}(t)=\text{cos}\,t\,\text{i}+\text{sin}\,t\,\text{j}.\) The derivative of this function is \({r}^{\prime }(t)=-\text{sin}\,t\,\text{i}+\text{cos}\,t\,\text{j}.\) If we substitute the value \(t=\pi \text{/}6\) into both functions we get

\[\text{r}(\frac{\pi }{6})=\frac{\sqrt{3}}{2}\,\text{i}+\frac{1}{2}\,\text{j}\,\text{and}\,{r}^{\prime }(\frac{\pi }{6})=-\frac{1}{2}\,\text{i}+\frac{\sqrt{3}}{2}\,\text{j}.\]

The graph of this function appears in Figure 1, along with the vectors \(\text{r}(\frac{\pi }{6})\) and \({r}^{\prime }(\frac{\pi }{6}).\)

Notice that the vector \({r}^{\prime }(\frac{\pi }{6})\) is tangent to the circle at the point corresponding to \(t=\pi \text{/}6.\) This is an example of a tangent vector to the plane curve defined by \(\text{r}(t)=\text{cos}\,t\,\text{i}+\text{sin}\,t\,\text{j}.\)

Definition

Let C be a curve defined by a vector-valued function r, and assume that \({r}^{\prime }(t)\) exists when \(t={t}_{0}.\) A tangent vector v at \(t={t}_{0}\) is any vector such that, when the tail of the vector is placed at point \(\text{r}({t}_{0})\) on the graph, vector v is tangent to curve C. Vector \({r}^{\prime }({t}_{0})\) is an example of a tangent vector at point \(t={t}_{0}.\) Furthermore, assume that \({r}^{\prime }(t)\ne \,0.\) The principal unit tangent vector at t is defined to be

\[\text{T}(t)=\frac{{r}^{\prime }(t)}{\Vert \,{r}^{\prime }(t)\Vert },\]

provided \(\Vert \,{r}^{\prime }(t)\Vert \ne 0.\)

The unit tangent vector is exactly what it sounds like: a unit vector that is tangent to the curve. To calculate a unit tangent vector, first find the derivative \({r}^{\prime }(t).\) Second, calculate the magnitude of the derivative. The third step is to divide the derivative by its magnitude.

Example 4

  • \(\begin{array}{llll}\text{First step:} & {r}^{\prime }(t) & = & -sin\,t\,\text{i}+\text{cos}\,t\,\text{j} \\ \text{Second step:} & \Vert \,{r}^{\prime }(t)\Vert & = & \sqrt{{(\text{-}\text{sin}\,t)}^{2}+{(\text{cos}\,t)}^{2}}=1 \\ \text{Third step:} & \text{T}(t) & = & \frac{{r}^{\prime }(t)}{\Vert \,{r}^{\prime }(t)\Vert }=\frac{\text{-sin}\,t\,\text{i}+\text{cos}\,t\,\text{j}}{1}=\text{-sin}\,t\,\text{i}+\text{cos}\,t\,\text{j}\end{array}\)

Integrals of Vector-Valued Functions

We introduced antiderivatives of real-valued functions in Antiderivatives and definite integrals of real-valued functions in The Definite Integral. Each of these concepts can be extended to vector-valued functions. Also, just as we can calculate the derivative of a vector-valued function by differentiating the component functions separately, we can calculate the antiderivative in the same manner. Furthermore, the Fundamental Theorem of Calculus applies to vector-valued functions as well.

The antiderivative of a vector-valued function appears in applications. For example, if a vector-valued function represents the velocity of an object at time t, then its antiderivative represents position. Or, if the function represents the acceleration of the object at a given time, then the antiderivative represents its velocity.

Definition

Let f, g, and h be integrable real-valued functions over the closed interval \([a,b].\)

  • The indefinite integral of a vector-valued function \(\text{r}(t)=f(t)\,\text{i}+g(t)\,\text{j}\) is

    The definite integral of a vector-valued function is

    \[∫[f(t)\,\text{i}+g(t)\,\text{j}]dt=[\,∫f(t)dt]\,\text{i}+[\,∫g(t)dt]\,\text{j}.\]

    \[{∫}_{a}^{b}[f(t)\,\text{i}+g(t)\,\text{j}]dt=[\,{∫}_{a}^{b}f(t)dt]\,\text{i}+[\,{∫}_{a}^{b}g(t)dt]\,\text{j}.\]

  • The indefinite integral of a vector-valued function \(\text{r}(t)=f(t)\,\text{i}+g(t)\,\text{j}+h(t)\,\text{k}\) is

    The definite integral of the vector-valued function is

    \[∫[f(t)\,\text{i}+g(t)\,\text{j}+h(t)\,\text{k}]dt=[\,∫f(t)dt]\,\text{i}+[\,∫g(t)dt]\,\text{j}+[\,∫h(t)dt]\,\text{k}.\]

    \[{∫}_{a}^{b}[f(t)\,\text{i}+g(t)\,\text{j}+h(t)\,\text{k}]dt=[\,{∫}_{a}^{b}f(t)dt]\,\text{i}+[\,{∫}_{a}^{b}g(t)dt]\,\text{j}+[\,{∫}_{a}^{b}h(t)dt]\,\text{k}.\]

Since the indefinite integral of a vector-valued function involves indefinite integrals of the component functions, each of these component integrals contains an integration constant. They can all be different. For example, in the two-dimensional case, we can have

\[∫f(t)dt=F(t)+{C}_{1}\,\text{and}\,∫g(t)dt=G(t)+{C}_{2},\]

where F and G are antiderivatives of f and g, respectively. Then

\[\begin{array}{ll}∫[f(t)\,\text{i}+g(t)\,\text{j}]dt & =[\,∫f(t)dt]\,\text{i}+[\,∫g(t)dt]\,\text{j} \\ & =(F(t)+{C}_{1})\,\text{i}+(G(t)+{C}_{2})\,\text{j} \\ & =F(t)\,\text{i}+G(t)\,\text{j}+{C}_{1}\,\text{i}+{C}_{2}\,\text{j} \\ & =F(t)\,\text{i}+G(t)\,\text{j}+\,\text{C},\end{array}\]

where \(\text{C}={C}_{1}\,\text{i}+{C}_{2}\,\text{j}.\) Therefore, the integration constant becomes a constant vector.

Example 5

Calculate each of the following integrals:

  • \(∫[(3{t}^{2}+2t)\,\text{i}+(3t-6)\,\text{j}+(6{t}^{3}+5{t}^{2}-4)\,\text{k}]\,dt\)
  • \(∫[\langle t,{t}^{2},{t}^{3}\rangle \,\times \,\langle {t}^{3},{t}^{2},t\rangle ]\,dt\)
  • \({∫}_{0}^{\pi \text{/}3}[\text{sin}\,2t\,\text{i}+\text{tan}\,t\,\text{j}+{e}^{-2t}\,\text{k}]\,dt\)

Integrate each component function separately (computing the cross product first in part b), then combine into a vector — remember the constant of integration is itself a vector for the indefinite integrals.

  • We use the first part of the definition of the integral of a space curve:

    \[\begin{array}{l}∫[(3{t}^{2}+2t)\,\text{i}+(3t-6)\,\text{j}+(6{t}^{3}+5{t}^{2}-4)\,\text{k}]dt \\ \\ \,=[\,∫3{t}^{2}+2t\,dt]\,\text{i}+[\,∫3t-6dt]\,\text{j}+[\,∫6{t}^{3}+5{t}^{2}-4dt]\,\text{k} \\ \,=({t}^{3}+{t}^{2})\,\text{i}+(\frac{3}{2}{t}^{2}-6t)\,\text{j}+(\frac{3}{2}{t}^{4}+\frac{5}{3}{t}^{3}-4t)\,\text{k}+\,\text{C}.\end{array}\]

  • First calculate \(\langle t,{t}^{2},{t}^{3}\rangle \,\times \,\langle {t}^{3},{t}^{2},t\rangle \text{:}\)

    Next, substitute this back into the integral and integrate:

    \[\begin{array}{ll}\langle t,{t}^{2},{t}^{3}\rangle \,\times \,\langle {t}^{3},{t}^{2},t\rangle & =|\begin{array}{lll}\text{i} & \text{j} & \text{k} \\ t & {t}^{2} & {t}^{3} \\ {t}^{3} & {t}^{2} & t\end{array}| \\ & =({t}^{2}(t)-{t}^{3}({t}^{2}))\,\text{i}-({t}^{2}-{t}^{3}({t}^{3}))\,\text{j}+(t({t}^{2})-{t}^{2}({t}^{3}))\,\text{k} \\ & =({t}^{3}-{t}^{5})\,\text{i}+({t}^{6}-{t}^{2})\,\text{j}+({t}^{3}-{t}^{5})\,\text{k}.\end{array}\]

    \[\begin{array}{ll}∫[\langle t,{t}^{2},{t}^{3}\rangle \,\times \,\langle {t}^{3},{t}^{2},t\rangle ]dt & =∫({t}^{3}-{t}^{5})\,\text{i}+({t}^{6}-{t}^{2})\,\text{j}+({t}^{3}-{t}^{5})\,\text{k}\,dt \\ & =(\frac{{t}^{4}}{4}-\frac{{t}^{6}}{6})\,\text{i}+(\frac{{t}^{7}}{7}-\frac{{t}^{3}}{3})\,\text{j}+(\frac{{t}^{4}}{4}-\frac{{t}^{6}}{6})\,\text{k}+\,\text{C}.\end{array}\]

  • Use the second part of the definition of the integral of a space curve:

    \[\begin{array}{l}{∫}_{0}^{\pi \text{/}3}[\text{sin}\,2t\,\text{i}+\text{tan}\,t\,\text{j}+{e}^{-2t}\,\text{k}]dt \\ \\ =[\,{∫}_{0}^{\pi \text{/}3}\text{sin}\,2tdt]\,\text{i}+[\,{∫}_{0}^{\pi \text{/}3}\text{tan}\,tdt]\,\text{j}+[\,{∫}_{0}^{\pi \text{/}3}{e}^{-2t}dt]\,\text{k} \\ ={(-\frac{1}{2}\text{cos}\,2t)|}_{0}^{\pi \text{/}3}\,\text{i}-{(\text{ln}(\text{cos}\,t))|}_{0}^{\pi \text{/}3}\,\text{j}-{(\frac{1}{2}{e}^{-2t})|}_{0}^{\pi \text{/}3}\,\text{k} \\ =(-\frac{1}{2}\text{cos}\,\frac{2\pi }{3}+\frac{1}{2}\text{cos}\,0)\,\text{i}-(\text{ln}(\text{cos}\,\frac{\pi }{3})-\text{ln}(\text{cos}\,0))\,\text{j}-(\frac{1}{2}{e}^{-2\pi \text{/}3}-\frac{1}{2}{e}^{-2(0)})\,\text{k} \\ =(\frac{1}{4}+\frac{1}{2})\,\text{i}-(\text{-}\text{ln}\,2)\,\text{j}-(\frac{1}{2}{e}^{-2\pi \text{/}3}-\frac{1}{2})\,\text{k} \\ =\frac{3}{4}\,\text{i}+(\text{ln}\,2)\,\text{j}+(\frac{1}{2}-\frac{1}{2}{e}^{-2\pi \text{/}3})\,\text{k}.\end{array}\]

Key Concepts

  • To calculate the derivative of a vector-valued function, calculate the derivatives of the component functions, then put them back into a new vector-valued function. (See Example 1 and Example 2.)
  • Many of the properties of differentiation from the Introduction to Derivatives also apply to vector-valued functions. (See Example 3.)
  • The derivative of a vector-valued function \(\text{r}(t)\) is also a tangent vector to the curve. The unit tangent vector \(\text{T}(t)\) is calculated by dividing the derivative of a vector-valued function by its magnitude. (See Example 4.)
  • The antiderivative of a vector-valued function is found by finding the antiderivatives of the component functions, then putting them back together in a vector-valued function. (See Example 5.)
  • The definite integral of a vector-valued function is found by finding the definite integrals of the component functions, then putting them back together in a vector-valued function. (See Example 5.)

Glossary

definite integral of a vector-valued function
the vector obtained by calculating the definite integral of each of the component functions of a given vector-valued function, then using the results as the components of the resulting function
derivative of a vector-valued function
the derivative of a vector-valued function \(\text{r}(t)\) is \({r}^{\prime }(t)=\underset{\Delta t\to 0}{\text{lim}}\frac{\text{r}(t+\Delta t)-\text{r}(t)}{\Delta t},\) provided the limit exists
indefinite integral of a vector-valued function
a vector-valued function with a derivative that is equal to a given vector-valued function
principal unit tangent vector
a unit vector tangent to a curve C
tangent vector
to \(\text{r}(t)\) at \(t={t}_{0}\) any vector v such that, when the tail of the vector is placed at point \(\text{r}({t}_{0})\) on the graph, vector v is tangent to curve C
